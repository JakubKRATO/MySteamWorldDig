REWARD SYSTEM

Odmeny dostaneš za každý run len RAZ. (Ak si uložíš hru a dohráš to znova tak nedostaneš rewardy.)


MONEY = peniaze ktoré zarobíš celkovo za run

TNT = koľko tnt blokov si použil za hru.


MONEY REWARD

AK MONEY - 1000 < 1000 : XP je 2, inak dostaneš 1 XP za každých 100 ktorých ostalo. (za každú začatú stovku)

900 MONEY => 2XP

1820 MONEY => 8XP

1799 MONEY => 7XP


TIME REWARD

Za každých začatých 20minút playtime dostaneš 1 XP.

TNT REWARD

TNT = počet TNT blokov, ktoré použiješ počas hry.

Čím menej TNT použiješ, tým väčšiu odmenu dostaneš.


Ak použiješ menej ako 70 TNT → dostaneš 5 bodov.

Ak použiješ 70–99 TNT → dostaneš 4 body.

Ak použiješ 100–124 TNT → dostaneš 3 body.

Ak použiješ 125–149 TNT → dostaneš 2 body.

Ak použiješ 150 alebo viac TNT → dostaneš 1 bod.


XP = MONEY REWARD + TIME REWARD + TNT REWARD

Coins je len náhodné číslo od 3 po 5. Dostaneš za každý dokončený run iba raz
